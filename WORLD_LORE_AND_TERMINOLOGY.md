# WORLD LORE AND TERMINOLOGY / 世界观与术语规范

> 本文件定义《万象仙乡》原创化阶段的公开世界观基线与命名规则。内部兼容字段不等于公开设定，不应直接显示给玩家。

## 一、世界观基线

《万象仙乡》由下界灵乡、修行道统与飞升仙域共同构成。玩家继承一处灵田家业，从耕作、采集、百艺和集民关系起步，逐渐接触地脉异象、宗门道统与仙界裂隙，最终在凡俗经营和长生修行之间建立自己的传承。

- **下界核心聚落：万象集** — 集民生活、委托、人情往来和基础商贸的中心。
- **核心家业：万象灵田洞天** — 玩家经营的灵田空间，承载种植、育种与土地养成。
- **主要水系：星澜河** — 节庆、垂钓与集民记忆的重要场所。
- **主要商圈：万象云集** — 灵材、日用物资和阶段性交易汇聚之处。
- **修行脉络：地脉—道统—飞升—仙域** — 从下界修行逐步进入独立仙界地图与高阶玩法。

## 四、叙事风格

1. 生活线以温暖、克制的东方乡野叙事为主，不直接复刻其他农场游戏的人物关系与节庆表达。
2. 修仙线强调“经营积累形成修行底蕴”，避免只用通用修仙名词堆砌。
3. 仙界线保持独立地图、独立界面和高阶资源循环，不与下界按钮混排。
4. 人物称谓、地点和事件优先服务于《万象仙乡》自身设定；新增文本应先查本规范。
5. 星澜河叙事强调公共水域、鱼群更新、护岸与巡汛责任；稀有渔获不天然等于应捕获。
6. 玄矿幽脉叙事强调测绘、支护、闭井信号、返程余力与开采边界；不把冒险冲动包装为专业勇敢。

## 五、兼容约束

为保证现有玩家云存档可继续读取，以下内部名称暂时保留：

- `taoyuan_*` localStorage/API 兼容键；
- 既有任务、NPC、道具、成就和地图 ID；
- 数据库名、部署目录、PM2 进程名与历史 API 字段；
- 历史更新日志中对当时版本的事实记录。

这些名称不得作为新玩家可见文案继续扩散。后续如迁移内部字段，必须先实现双读、双写或显式迁移层。
